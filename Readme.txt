这是一个简单的塔防小游戏，包含3种防御塔以及一种敌人
金矿：4点hp，花费5金币，每5秒生产5金币
炮塔：6点hp，花费10金币，每1秒发射一颗3伤害的炮弹
盾墙：16点hp，花费5金币
敌人：10点hp，每1秒前进1格，每1秒可造成3点伤害
初始共10点生命10金币，每有一个敌人越过底边界扣除1点生命，每有一个敌人被消灭获得1点分数
生命为0或游戏时间到达180秒时结束游戏，最终得分为剩余生命与分数的乘积

包含OTHER、TOWER、ENEMY、GAME共4个命名空间，其中
OTHER包含1个Point类，用于处理位置信息
TOWER包含1个基类Tower及其3个派生类Generator、Attacker、Defender以及一个额外的Bullet类，用于处理防御塔行为
ENEMY包含1个Enemy类，用于处理敌人行为
GAME包含1个Game类，用于处理游戏逻辑
